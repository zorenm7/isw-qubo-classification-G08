```text
### Implementation Details & Documentation

* **Optimization Algorithm:** I've opted for Simulated Annealing using the open-source `dwave-neal` library. It's highly effective for heuristic optimization on dense QUBO matrices without requiring actual quantum hardware access during development.
* **Mathematical Formulation:** The objective function minimizes $f(x) = -[\alpha \sum x_j |\rho_{Vj}| - (1-\alpha) \sum\sum x_j x_k |\rho_{jk}|]$. Translating this to standard QUBO form ($\min \sum_i Q_{ii} x_i + \sum_{i<j} Q_{ij} x_i x_j$), the diagonal terms incorporate both the influence (Spearman correlation with the target) and the $j=k$ dependence self-correlation. The off-diagonal terms encapsulate the $2(1-\alpha)|\rho_{jk}|$ cross-correlations.
* **Alpha Search:** A binary search strategy is implemented across the [0, 1] domain of $\alpha$. This allows the algorithm to dynamically find the optimal penalty factor that yields exactly $K \pm allowance$ features, avoiding an inefficient linear scan while bounding iterations to `alpha_computations`.

```
